# Markdown 文件查看器

本项目是一个本地运行的 Vue3 Markdown 文件查看器。启动后会自动扫描最近 3 天创建的 Markdown 文件，默认忽略 C 盘，左侧按“盘符 / Markdown 所在文件夹 / 文件”的结构展示，右侧渲染当前选中文件内容。

## 运行

```bash
npm install
npm run dev
```

打开 Vite 输出的本地地址，默认是：

```text
http://127.0.0.1:5173
```

## 扫描说明

- 默认自动扫描当前电脑可访问的磁盘，但会忽略 C 盘。
- 默认只展示最近 3 天创建的 Markdown 文件。
- 点击“选择”可以在弹窗目录树中选择要扫描的盘符或文件夹，弹窗只展示存在 Markdown 文件的路径。
- 手动选择某个盘符或文件夹后，会扫描该路径下所有时间的 Markdown 文件。
- 左侧目录树会保留最近 5 个打开过的 Markdown 文件，刷新页面后仍会显示。
- 点击“全部磁盘”会扫描当前电脑可访问的非 C 盘磁盘根目录。
- 左侧不会展开完整多级目录，每个 Markdown 文件会归入它的直接父文件夹，文件下方显示完整路径。
- 为避免扫描过慢，服务端会跳过 `node_modules`、`.git`、`Windows`、`AppData` 等常见系统或缓存目录。
- 默认最多返回 20000 个 Markdown 文件，单个预览文件最大 5 MB。
